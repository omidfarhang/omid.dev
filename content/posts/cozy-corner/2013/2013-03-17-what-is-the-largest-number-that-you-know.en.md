Tonight I was talking with a friend about [bad news for Google](https://techcrunch.com/2013/03/17/god-damn-it-google/) when he asked, almost as a side note:

> Did you know *googol* is a number?

I did not. I knew Google, not googol. So I went looking — and fell into the oddly satisfying rabbit hole of **named large numbers**.

What is the largest number *you* know by name?

## Short scale vs long scale

Before the big names, a quiet trap: the same word can mean different powers of ten depending on where you learned arithmetic.

| Name | Short scale (US, Canada, modern British) | Long scale (continental Europe, older British) |
|:-----|:-----------------------------------------|:-----------------------------------------------|
| Million | 10<sup>6</sup> | 10<sup>6</sup> |
| Milliard | 10<sup>9</sup> | — |
| Billion | 10<sup>9</sup> | 10<sup>12</sup> |
| Trillion | 10<sup>12</sup> | 10<sup>18</sup> |
| Quadrillion | 10<sup>15</sup> | 10<sup>24</sup> |
| Quintillion | 10<sup>18</sup> | 10<sup>30</sup> |
| Sextillion | 10<sup>21</sup> | 10<sup>36</sup> |
| Septillion | 10<sup>24</sup> | 10<sup>42</sup> |
| Octillion | 10<sup>27</sup> | 10<sup>48</sup> |
| Nonillion | 10<sup>30</sup> | 10<sup>54</sup> |
| Decillion | 10<sup>33</sup> | 10<sup>60</sup> |
| Undecillion | 10<sup>36</sup> | 10<sup>66</sup> |
| Duodecillion | 10<sup>39</sup> | 10<sup>72</sup> |
| Tredecillion | 10<sup>42</sup> | 10<sup>78</sup> |
| Quattuordecillion | 10<sup>45</sup> | 10<sup>84</sup> |
| Quindecillion | 10<sup>48</sup> | 10<sup>90</sup> |
| Sexdecillion | 10<sup>51</sup> | 10<sup>96</sup> |
| Septendecillion | 10<sup>54</sup> | 10<sup>102</sup> |
| Octodecillion | 10<sup>57</sup> | 10<sup>108</sup> |
| Novemdecillion | 10<sup>60</sup> | 10<sup>114</sup> |
| Vigintillion | 10<sup>63</sup> | 10<sup>120</sup> |
| Centillion | 10<sup>303</sup> | 10<sup>600</sup> |

On the **short scale**, each new *-illion* is a thousand times the previous one (billion = thousand million, trillion = thousand billion, …). On the **long scale**, each new *-illion* is a *million* times the previous one — so a European “billion” is what Americans call a trillion.

That alone explains a surprising number of confused news headlines about government budgets.

Apart from *million*, these *-illion* words come from Latin-ish prefixes (*bi-*, *tri-*, *quadri-*, …) stuck onto the stem *-illion*. *Centillion* is usually the highest *-illion* you’ll find in mainstream dictionaries. You can keep inventing names (*trigintillion*, *unvigintillion*, …), but that does not make them “official” — they’re pattern extensions, not common vocabulary.

## Googol and googolplex

Then there are the celebrity numbers that aren’t *-illions* at all:

| Name | Value |
|:-----|:------|
| Googol | 10<sup>100</sup> |
| Googolplex | 10<sup>googol</sup> |

A **googol** is a 1 followed by **100 zeros**. Edward Kasner popularized the name in the 1930s (story goes his young nephew Milton Sirotta coined it). A **googolplex** is a 1 followed by a *googol* zeros — a number so large that writing it out in ordinary decimal digits is physically impossible in our universe. There aren’t enough atoms for the zeros.

A few reference points, just for scale:

- Atoms in the observable universe: on the order of 10<sup>80</sup> — already well *below* a googol.
- A googol: 10<sup>100</sup> — bigger than the atom count, still “writable” if you’re patient.
- A googolplex: 10<sup>10<sup>100</sup></sup> — you cannot write it down. Not even close.

Dictionaries generally include *googol* and *googolplex* (crediting Kasner / Newman / the nephew). They usually stop there — names like *googolduplex* exist mostly as curious fan extensions. The *Oxford English Dictionary* notes that *googol* and *googolplex* are “not in formal mathematical use.” They’re names for intuition, not tools you write proofs with.

And yes: **Google** the company is a deliberate misspelling of *googol* — a joke about indexing a huge amount of information that became the most famous brand on the web. Which is how a math-trivia night ended up tangled with company news in the first place.

## Is there a largest number?

No. For any named finite number *N*, *N* + 1 is larger. You can always go bigger.

There are named numbers that dwarf googolplex — [Graham’s number](https://en.wikipedia.org/wiki/Graham%27s_number) is a classic example from combinatorial mathematics — but the deeper point is simpler: **names run out long before the integers do**. Language is a tiny island in an ocean of unnameable sizes.

So maybe the better question isn’t “what’s the largest number you know?” but “how far does your mental map of size go — and where does it stop feeling real?”

Mine used to stop somewhere around “billion.” Tonight it briefly included googolplex. Tomorrow it’ll probably shrink back to whatever fits in a bank statement — and that’s fine. The fun is knowing the cliff is there.
